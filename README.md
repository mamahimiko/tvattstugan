# Tvättstugan Bokningtavla

A personal booking memo app inspired by the traditional laundry room reservation boards found in Swedish apartment buildings.

Users can select and save their booked laundry date and time slot. Once the booking button is pressed, the selected reservation is stored in localStorage, allowing users to revisit their booking anytime.

## Features

* Traditional Swedish laundry board inspired UI
* Save booked date and time locally
* Persistent data using localStorage
* Responsive layouts for desktop and mobile
* Interactive booking and cancel interactions

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* React Icons

## Inspiration

This project was inspired by the analog laundry booking boards commonly seen in Swedish apartment shared laundry rooms (Tvättstuga).
